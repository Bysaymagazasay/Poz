# BYSAY Poz ve Maliyet Programı

2026 Temmuz mekanik tesisat birim fiyat listesindeki pozları sorgulayan, Excel/CSV/Word dosyalarından poz ve miktar aktarabilen web uygulaması.

## Özellikler
- Poz numarasıyla tanım, birim ve fiyat sorgulama
- Çoklu satır girişi
- Dosyadan poz ve miktar içe aktarma
- Miktar × birim fiyat satır toplamı
- Genel toplam hesaplama
- CSV dışa aktarma

Programın ana sayfası `index.html` dosyasıdır.
